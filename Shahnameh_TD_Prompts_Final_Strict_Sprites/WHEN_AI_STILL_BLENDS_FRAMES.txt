WHEN AN IMAGE MODEL STILL BLENDS FRAMES

Some image generators are weak at exact technical sprite sheets. The prompts are now strict, but no image model can be guaranteed to obey every grid instruction every time.

If a generator still creates blended animation strips:
1. Generate the base character first.
2. Generate one frame at a time using a 256x256 canvas and pure #00FF00 background.
3. Keep the same reference image, costume, scale, and bottom-center pivot for every frame.
4. Assemble the individual frames into the horizontal strip in an image editor or Godot import workflow.

Reject any output with pale green, fog, motion ghosts, overlapping frames, decorative stars, or a continuous scene background.
