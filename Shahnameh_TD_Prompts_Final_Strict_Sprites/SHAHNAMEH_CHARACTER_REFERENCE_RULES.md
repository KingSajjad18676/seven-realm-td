# Shahnameh Character Reference Rules

This update was created to fix a specific problem: some generated characters looked like **generic fantasy warriors** instead of characters from the **Shahnameh / Persian epic world**.

## Mandatory rule
Every **character-related prompt** in this pack now includes a dedicated Shahnameh-reference block.

Covered prompt groups:
- character base designs
- character animation strips
- enemies and bosses
- companions
- portraits and codex character art

## What the new prompts force
- Persian / Iranian heroic costume language
- Shahnameh manuscript and Persian miniature inspiration
- Rostam-specific markers for Rostam assets
- Persian mythic identity for beasts and enemies
- stronger anti-generic negative prompts

## Character prompt count updated
- Character-like prompts updated: 256 files
- Character-like asset IDs covered: 256

## Reminder about backgrounds
- Isolated sprites / sprite sheets: green background `#00FF00`
- Maps, layers, cards, portraits, UI, illustrations: full canvas, no forced green background unless the asset is an isolated sprite
