# Cursor Prompt — Roguelite Expedition Prototype

Create a small roguelite expedition prototype after the core battle loop works.

Features:

- branching map with 5 nodes
- node types: battle, elite, merchant, shrine, boss
- after battle, choose 1 of 3 temporary blessings
- blessings modify tower damage, hero cooldown, or starting gold
- final boss node ends run and grants reward

Use ScriptableObjects for node data and blessing data.

Keep it separate from campaign progression.
